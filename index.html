周易摇卦 Prompt 生成器

这个版本不直接调用 DeepSeek API，而是：你在这里摇卦、选择问题方向、填写出生信息与具体问题，系统自动生成一段高质量的解卦提示词。
然后你点击“一键复制 Prompt”，直接粘贴到 DeepSeek、ChatGPT 或别的 AI 里，就能拿到解卦分析。

1）配置与起卦

推荐流程：先点“摇一次骰子” → 看本卦和变卦 → 生成 Prompt → 一键复制 → 粘到 DeepSeek 网页版。

使用方式：
1. 在这个页面里完成摇卦和问题填写。
2. 点击“生成 Prompt”。
3. 点击“复制 Prompt”。
4. 打开 DeepSeek 网页版，把它粘进去即可。

这个版本的优点是：免费、稳定、不暴露 API Key，也适合先做产品验证。
